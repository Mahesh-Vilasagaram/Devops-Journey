## Key Concepts
- VPC is an isolated virtual network in AWS
- Public subnets have internet access
- Private subnets do not have direct internet access

## Security Groups
- Act as virtual firewalls
- Control inbound and outbound traffic
- Are stateful

## Hands-on
- Modified security group rules
- Allowed SSH and HTTP access
- Verified traffic flow to EC2

## Interview Notes
- Security Groups work at instance level
- Only allowed traffic is permitted
- NAT Gateway is used for private subnet internet access
