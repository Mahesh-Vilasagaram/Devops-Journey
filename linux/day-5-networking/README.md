# Linux Day 5 – Networking Basics

## Commands Practiced
- ip a
- ss -tuln
- ping
- nslookup
- curl

## Key Learnings
- IP identifies a machine, port identifies an application
- DNS converts domain names to IP addresses
- curl is used to test HTTP responses

## Interview Scenarios
- Ping works but website down → check service and ports
- curl works locally but browser fails → check firewall and binding
