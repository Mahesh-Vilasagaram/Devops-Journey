## Commands Practiced
- ls -l
- chmod 755
- chmod 644
- chmod u+x
- chown user:user

## Key Learnings
- Linux permissions control access to files
- Owner, Group, Others define who can do what
- Numeric permissions use r=4, w=2, x=1

## Interview Notes
- 755 is commonly used for scripts and directories
- 644 is used for normal files
- 777 is insecure and should be avoided in production
